13 [PantaRei\Bundle\BootstrapBundle](https://github.com/pantarei/bootstrap-bundle)
14 is a collection of code to integrate [Twitter
15 Bootstrap](http://twitter.github.com/bootstrap/) as easy as possible
16 into your [Symfony2](http://www.symfony.com) Project.
17 
18 Installation
19 ------------
20 
21 First you need to add `pantarei/bootstrap-bundle` to `composer.json` (in
22 the Symfony directory ) :
23 
24  {
25  "require": {
26  "pantarei/bootstrap-bundle": "3.0.*@dev"
27  }
28  }
29 
30 and execute the command `composer update`:
31 
32 You also have to add `BootstrapBundle` to your `AppKernel.php` (in the
33 symfony/app ):
34 
35  class AppKernel extends Kernel
36  {
37  public function registerBundles()
38  {
39  $bundles = array(
40  new PantaRei\Bundle\BootstrapBundle\BootstrapBundle();
41  );
42  return $bundles;
43  }
44  }
45 
46 and execute the command `composer update`:

61 ### Without Assetic
62 
63 Create symlink for the asset files from the `vendor/twitter` directory
64 into your web directory (Your "Symfony" folder):
65 
66  cd web/bundles
67  ln -s ../../vendor/twitter twitter
68 
69 [ Below are already in file, change if need. ] Now you can include
70 boostrap css and js in main template:
71 
72  <link rel="stylesheet" href="{{ asset('bundles/twitter/bootstrap/docs/assets/css/bootstrap.css') }}">
73  <script src="{{ asset('bundles/twitter/bootstrap/docs/assets/js/jquery.js') }}"></script>
74  <script src="{{ asset('bundles/twitter/bootstrap/docs/assets/js/bootstrap.min.js') }}"></script>
75 
76 ### With Assetic
77 
78 If you want to use LessPHP to compile the Bootstrap LESS files, you need
79 update your `composer.json` file and execute the following line:
80 `composer update`:
81 
82  {
83  "require": {
84  "leafo/lessphp": "0.3.9"
85  }
86  }
87 
88 Now change your `app/config/config.yml` to this:
89 
90  # Assetic Configuration
91  assetic:
92  filters:
93  lessphp:
94  file: %kernel.root_dir%/../vendor/leafo/lessphp/lessc.inc.php
95  apply_to: "\.less$"
96 
97 After that, the last thing we need is to include bootstrap in main
98 template:
99 
100  {% stylesheets
101  'bundles/twitter/bootstrap/less/*.less'
102  %}
103  <link rel="styleshet" href="{{ asset_url }} "/>
104  {% endstylesheets %}

106 Examples
107 --------
108 
109 If you hope to enable the examples as reference, update your
110 `app/config/routing.yml` file to this:
111 
112  pantarei_bootstrap:
113  resource: "@BootstrapBundle/Resources/config/routing.yml"
114  prefix: /_bootstrap
115 
116 Then you can access `_bootstrap/starter-template` or other pages as
117 example.

166 Customize CSS
167 -------------
168 
169 Go to Symfony directory in ssh :
170 
171 php app/console assets:install web --symlink
172 
173 then add the command below in the composer.json :
174 
175  "minimum-stability": "alpha",
176  "extra": {
177  "symfony-assets-install": "symlink"
178  }
179 
180 and execute the command `composer update`:
181 
182 Create cusotom.css file in [yourdirectory]/Resources/public/css: Then
183 add below line into ANY .html.twig file to run your custom css :
184 
185 {% block styles %}
186 <link rel="stylesheet" href="{{ asset('bundles/yourdirectory/css/custom.css') }}">
187 {% endblock %}
188 
189 then the CSS file will auto asset from your own bundle public and make
190 it work in your own page.
